Instructions

The code is a sequence of colours. Try to solve the code based on the results of each guess.

Choose the length of the code and the number of possible colours for each square from the menus in the upper left corner of your screen. Then, click New Game to create the code and begin guessing.

Enter your guess by clicking the coloured squares in the row at the bottom of the screen. Once the row above is filled, click Submit Guess to find out how close you guess is to the solution.

Each guess you submit will be shown from the top of the screen, with the most recent guess shown highest. For each correct colour in the correct spot in the sequence, a black box will be displayed to the left of your guess. For each correct colour in the incorrect order, a white box will be displayed.
